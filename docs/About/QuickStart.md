# 快速开始

### 感谢您使用 StarLight_Core

## 1.下载

在 [Github Packages](https://github.com/orgs/Ink-Marks-Studio/packages?repo_name=StarLight.Core) 中下载，

或直接引用 [NuGet 程序包](https://www.nuget.org/packages/StarLight_Core) .

## 2.引用

在C#中使用using引用核心的类:

```csharp
using StarLight_Core;
```
