## 简介

一个简单,模块化,全能的启动器核心,

快速启动、下载游戏核心、还有各种封装好的小工具!

StarLight.Core 几乎支持.NET的受支持的版本,包括.NET Core 6~8.0,兼容性强大.

## 支持列表
* 1.启动游戏
* 2.获取核心版本列表
* 3.获取Java版本列表
* 4.离线、正版和第三方验证器
* 5.下载核心本体文件
* 6.下载模组加载器(开发中)
